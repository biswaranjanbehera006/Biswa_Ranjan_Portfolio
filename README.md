# 🌐 Animated Portfolio Website

A sleek, responsive, and fully customizable **Animated Portfolio Website** built using **React.js** and **Tailwind CSS**. This project gained **27K+ views on YouTube** — loved by developers and students for its **modern design**, **clean code**, and **developer-friendly structure**.

- ⚛️ **React.js** — Frontend JavaScript Library

## 🧑‍💻 Quick Start Guide

### Step 1: Clone the Repository

```bash
git clone https://github.com/biswaranjanbehera006/Biswa_Ranjan_Portfolio.git
cd Biswa_Ranjan_Portfolio
````

### Step 2: Install Dependencies

```bash
npm install
```

### Step 3: Run the Development Server

```bash
npm start
```

### Step 4: Build for Production

```bash
npm run build
```
